# NAME

lms - Let Me See command

# VERSION

Version 0.03

# SYNOPSIS

lms command/library

# DESCRIPTION

This is a small program to look a command or library file.

Only Perl is supported now.

# INSTALL

cpanm https://github.com/kaz-utashiro/App-lms.git
